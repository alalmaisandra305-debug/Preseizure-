PreSeizure — Wearable AI Seizure Prediction

Predicting epileptic seizures minutes before onset using PPG heart-rate variability and motion sensors.

Research Project

Overview

PreSeizure is a high-school research project by Sandra Ali Abdulhadi Alalmai, a 16-year-old researcher. The project investigates whether wearable sensors and AI can provide early warning of epileptic seizures to improve safety and response time.

"I had so much fun working on this project — designing models, testing signals, and building prototypes felt like real science. I love it!" — Sandra

Methods

We use a combination of signal processing and machine learning:

  • Preprocessing: sliding windows across PPG and accelerometer signals
  • Feature extraction: HRV time-domain features (SDNN, RMSSD, pNN50) and motion statistics
  • Models: Random Forest baseline and 1D-CNN / LSTM deep models
  • Explainability: SHAP for model interpretability and feature importance
